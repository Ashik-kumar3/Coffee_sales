MATPLOTLIB
----------
Python package for data visualization.
Open source.
Inspired by Matlab.
Mainly uses matplotlib.pyplot.
Developed by John D. Hunter.
Open sourced(BSD licensed).
Object Oriented.
Publication type plots.

import matplotlib.pyplot as plt
plt.plot(x, y)
plt.show()

for plotting subplots
---------------------
fig, ax = plt.subplots()
ax.plot(x, y)
plt.show()



Different types of plots in MATPLOTLIB
--------------------------------------
Plotting
Function	      Purpose

plt.plot()	  -   Line plot
plt.scatter() -   Scatter plot
plt.bar()	  -   Bar chart
plt.hist()	  -   Histogram
plt.boxplot() -   Box plot
plt.pie()	  -   Pie chart
plt.imshow()  -   Display image
plt.subplot() -   Multiple plots


Customization
-------------
Function	     Purpose

plt.title()	  -  Title
plt.xlabel()  -  X label
plt.ylabel()  -  Y label
plt.legend()  -  Show legend
plt.grid()	  -  Add grid
plt.xlim()	  -  X-axis limit
plt.ylim()	  -  Y-axis limit
plt.xticks()  -  Customize ticks
plt.yticks()  -  Customize ticks

plt.tight_layout() - prevents overlapping
plt.savefig("plot.fig") - for saving plot
